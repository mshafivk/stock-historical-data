# stock-historical-data

## Pre-requisites
  - NodeJS
  - MongoDB
  
# server
 - cd server
 - npm install
 - npm run seed.js
 - npm start - to start the server
 
# UI
- cd stocks-history-ui
- npm install
- npm start
- web page will be loaded in http://localhost:3000
- Make sure server is running before checking search and other functionalities
